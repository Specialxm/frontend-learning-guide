## [快速开始](#快速开始)

### [环境要求](#环境要求)
- **Node.js**: >= 20.0.0
- **包管理器**: npm >= 8.0.0
- **浏览器**: 现代浏览器 (Chrome 90+, Firefox 88+, Safari 14+)

### [本地开发](#本地开发)
```bash
# 克隆项目
git clone https://github.com/Specialxm/frontend-learning-guide.git
cd frontend-learning-guide

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### [构建部署](#构建部署)
```bash
# 构建生产版本
npm run build

# 部署到GitHub Pages
npm run deploy

# 清理构建文件
npm run clean
```
